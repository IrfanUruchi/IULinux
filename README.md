# IULinux

Early development project.

Ubuntu-based KDE Linux distribution focused on development, AI, and workstation performance.

IULinux is based on Ubuntu and KDE Plasma, with planned support for developer tools, containers, AI tooling, NVIDIA setup, performance tuning, and a clean workstation UI.

The project is currently in early development. The first public alpha is planned for the end of November 2026.
